# Submission System Options

## Option 1: Google Form + Drive (Simplest)

**Pros:**
- Free
- Everyone knows how to use it
- Photo upload works
- Easy to share link

**Cons:**
- Recipe formatting will be messy
- Manual cleanup needed
- Not pretty

**Setup:** ~30 minutes

---

## Option 2: Typeform or Jotform (Polished)

**Pros:**
- Beautiful UX
- Better form logic
- Photo upload
- Exports nicely

**Cons:**
- May need paid tier for file uploads
- Another account to manage

**Setup:** ~1 hour

---

## Option 3: Custom Simple Site (Most Control)

## Recommendation

Start with **Google Form** to validate the process and get early submissions. If it takes off, we can build something nicer.

## Form Fields Needed

1. Parent name + email
2. Child's name
3. Child's age/grade
4. Recipe name
5. Ingredients (text area)
6. Instructions (text area)
7. Photo upload
8. Story/fun fact (text area)
9. Permission checkbox (photo release)

## Photo Requirements

- Minimum resolution: 1200x1200 px (for print)
- Accepted formats: JPG, PNG
- Max file size: 10MB
